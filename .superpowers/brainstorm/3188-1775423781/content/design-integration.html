// 1. Criar refs para cada slide
const slideRefs = slides.map(() => useRef<HTMLDivElement>(null))

// 2. Gerar filenames: "{slug}-{tipo}"
//    ex: "piloto-testes-cover", "piloto-testes-content"
const filenames = slides.map(s => `${carousel.slug}-${s.type}`)

// 3. Cada slide wrapper recebe seu ref
<div ref={slideRefs[i]} style={{ width, height, ... }}>
  {renderSlide(slide, ...)}
</div>

// 4. SlideExportControls renderizado abaixo de cada slide
<SlideExportControls
  ref={slideRefs[i]}
  filename={filenames[i]}
  allRefs={slideRefs}
  allFilenames={filenames}
/>

Limitação conhecida do html-to-image

Google Fonts no export: html-to-image não consegue embedar fontes carregadas via @import de CDN externo. Para garantir as fontes no PNG, o hook vai usar a opção fontEmbedCSS ou um preferredFontFormat. Alternativa: usar toBlob com espera de fontes via document.fonts.ready antes de capturar.
