Estrutura de arquivos
src/
├── features/
│   └── export/
│       ├── useExport.ts          ← hook: lógica de export (html-to-image + download)
│       └── SlideExportControls.tsx  ← componente: botões "Este slide" e "Todos"
└── App.tsx                       ← passa refs + filename para SlideExportControls

Responsabilidades

Arquivo Faz Não faz
useExport.ts Chama toPng(), dispara download, gerencia estado loading/error Renderiza UI, conhece slides
SlideExportControls.tsx Renderiza botões, mostra loading state, dispara toast Lógica de export, conhece html-to-image
App.tsx Cria refs para cada slide, passa filename baseado em slug+tipo Lógica de export, UI de botões

Fluxo de dados

Data flow
App.tsx
  └─ cria: refs[], filenames[] (ex: "piloto-testes-cover")
  └─ renderiza: <SlideExportControls ref={ref} filename={name} allRefs={refs} allFilenames={names} />
        └─ usa: useExport(ref, filename)
              └─ exportOne()  → toPng(ref) → triggerDownload(filename.png)
              └─ exportAll()  → loop refs → toPng(ref) → triggerDownload em sequência
